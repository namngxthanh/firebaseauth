## ğŸŒ Final Notes

* You **must eject Expo to use native modules** like Firebase in full capacity
* Don't forget to import and initialize Firebase in `firebaseConfig.js`
* Use Firebase Realtime rules to secure your database

Example:

```json
{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}
```
